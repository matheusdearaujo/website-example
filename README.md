# Website Example

### Requirements

- docker
- docker-compose

### Installation

1. Clone this repository:
```sh
git clone https://github.com/matheusdearaujo/website-example
```

2. Change directory.
```sh
cd website-example
```

**OBS: Is important that your env file is configured before start server.**

3. Run project with Docker Compose:

```sh
make
```
